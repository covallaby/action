
# Product

Covallaby is a GitHub-first coverage tool.

Primary deliverables:

- Sticky PR comments
- GitHub Step Summary
- HTML coverage report
- Patch coverage
- Project coverage
- Threshold enforcement
- LCOV, JaCoCo, Cobertura, xccov JSON
- Future hosted dashboard

Tagline:

Beautiful coverage reports for your pull requests.

## Demo note

Docs-only changes have no coverable lines, so Covallaby's patch gate
passes them without demanding tests for prose.
