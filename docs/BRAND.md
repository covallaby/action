
# Covallaby Brand

> The full brand kit — logos, icons, favicon, social preview, design tokens,
> and the canonical [brand guide](../brand/docs/BRAND_GUIDE.md) — lives in
> [`brand/`](../brand/). This file is the original spec it grew from.

## Mascot

A cute wallaby—not a kangaroo.

Characteristics:
- Rounded face
- Large expressive ears
- Big eyes
- Tiny smile
- Warm tan fur
- Emerald green accents
- Carries a coverage report in its pouch

## Logo

Simple vector mark.

Wallaby head inside a circle.

Works at:
- 16x16
- 32x32
- GitHub avatar
- favicon
- app icon

## Voice

Friendly.
Supportive.
Never shames developers.

Examples:

"You're covered."

"Nice jump! Patch coverage improved."

"4 uncovered lines need some love."
