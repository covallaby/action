# Covallaby Brand Guide

## Name

**Covallaby**

Pronunciation: `cov-WALL-uh-bee`

## Tagline

**Beautiful coverage reports for your pull requests.**

## Mascot

A cute simplified wallaby. It should read as small, friendly, approachable, and wallaby-like rather than kangaroo-like.

## Brand personality

Helpful, optimistic, calm, direct, a little cute, never shaming.

## Voice examples

- You're covered.
- Nice! Patch coverage improved.
- Almost covered.
- 3 uncovered lines need some love.
- Coverage is below the required threshold.

## Visual style

Use rounded cards, cream backgrounds, emerald accents, friendly wallaby marks, and clean GitHub-native UI.

Avoid enterprise dashboard vibes, complex mascot illustrations in tiny places, too many emojis, and visual clutter.
