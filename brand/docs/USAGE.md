# Asset Usage

## Primary files

- `svg/icon.svg` — primary app icon
- `svg/wallaby-mark.svg` — transparent mascot mark
- `svg/wordmark.svg` — large wordmark with tagline
- `svg/logo-horizontal.svg` — compact logo lockup
- `favicon/favicon.svg` — favicon
- `github/social-preview.svg` — GitHub/social preview banner

## Recommended usage

Use `svg/icon.svg` for GitHub avatar/app icon. Use `svg/logo-horizontal.svg` in README headers. Use `github/social-preview.svg` for the repo social preview.

The generated PNG branding boards in `mockups/` are inspiration only. Use the SVGs as the starter brand assets.
