# Covallaby Brand Direction

## Personality

Covallaby should feel:
- cute
- developer-friendly
- simple
- approachable
- polished
- open-source

It should not feel:
- enterprise
- childish
- overly mascot-driven
- complicated
- generic SaaS

## Visual direction

Use the PNG brand boards as the source of truth.

Preferred style:
- simple wallaby
- rounded shapes
- soft green / cream palette
- clean product UI
- easy-to-recognize mascot
- friendly but professional

## Colors

Approximate palette:

- Primary green: `#22C55E`
- Dark green: `#166534`
- Cream: `#F8F3E7`
- Warm tan: `#D9A066`
- Ink: `#111827`
- Muted text: `#6B7280`
- Border: `#E5E7EB`

## Copy

Good:
- You're covered.
- Almost covered.
- Nice! Patch coverage improved.
- 3 uncovered lines need some love.
- Beautiful coverage reports for your pull requests.

Avoid:
- Coverage failed badly.
- Your PR is bad.
- Enterprise-y policy language.
