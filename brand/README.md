# Covallaby PNG Brand Pack

This is the practical brand pack for Fable.

Use the generated PNG branding boards/mockups as the visual source of truth.

Do **not** recreate the logo as SVG yet.
Do **not** over-invest in mascot polish yet.
Do **not** block product development on branding.

## Brand direction

Name: **Covallaby**

Tagline:

> Beautiful coverage reports for your pull requests.

Mascot:

A cute, simple wallaby.

The best current direction is the clean/simple wallaby logo board from the generated PNGs. Treat it as inspiration for README, landing page, GitHub Marketplace, and HTML report styling.

## Product priority

The GitHub Action is the product.

Branding should support:
- README
- GitHub avatar
- Marketplace page
- HTML report
- PR comment personality

But do not spend days polishing the mascot before the Action works.
